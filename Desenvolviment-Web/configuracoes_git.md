## Configurações GIT

### Configurar usuário:
    - Comandos
        git config --global user.name "nome do usuario git"

        git config --global user.email  " "


### Comandos de repositorios

git pull
git add .
git commit -m ""
git push;
git push --set-upstream nome-branch

### Atualizar repositório da máquina

- Comando
    git pull
